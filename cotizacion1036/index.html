Marketing VIP® Entendimiento
Floow: el SaaS para empresas rotulistas

Analizamos en detalle el flujo del sistema —desde el onboarding hasta los tableros Kanban, CRM, gestión de proveedores, generación de proyectos con IA y dashboard master admin— para proponer una arquitectura backend robusta y veloz.

🔍 Lo que entendemos del proyecto

Floow es una plataforma SaaS multi-tenant diseñada específicamente para empresas de rotulación y señalética. A partir del flujo completo revisado, identificamos un sistema con múltiples módulos interconectados: gestión de empresas (sedes, miembros, roles y permisos), CRM embudo comercial tipo Trello, gestión de clientes y proveedores con OCR e IA, tableros de producción Kanban con sub-tableros, generación de proyectos asistida por IA (mockups, fotomontajes, renders), chat IA especializado en rotulación, sistema de facturación y presupuestos, y un dashboard Master Admin con analíticas avanzadas impulsadas por DeepSeek. El frontend ya está diseñado; el reto está íntegramente en construir el backend, la lógica de negocio, la base de datos y todas las APIs que lo sustenten.

01

Arquitectura multi-tenant

Cada empresa rotulista es un tenant independiente con sus propios miembros, sedes, proyectos y datos. Aislamiento seguro con Supabase (PostgreSQL) y Row Level Security.

02

APIs REST escalables

APIs bien estructuradas para todos los módulos: auth, empresas, CRM, proyectos, tableros, proveedores, clientes, facturación, notificaciones y analíticas.

03

Integraciones IA

Claude Haiku para chat Sign, DeepSeek para analíticas cron, OpenAI/Claude Vision para OCR de documentos, Flux.ai/Freepik para generación de mockups y renders.

04

Lógica de negocio compleja

Embudo comercial que se convierte automáticamente en tareas Kanban, descomposición IA de proyectos en subtareas, cron de analíticas, sistema de permisos por rol y área.

05

Integraciones de terceros

Stripe para suscripciones y planes, Brevo para notificaciones y emails transaccionales, APIs de precios de materiales en tiempo real.

06

Base de conocimientos + cron

Sistema KB alimentado con sugerencias de la IA, cron jobs que extraen y analizan datos diariamente para el Master Admin con sentimiento, señales y resúmenes.

Marketing VIP® Alcance
¿Qué incluye?

El alcance cubre el desarrollo backend completo de todos los módulos definidos en el flujo de Floow. El frontend ya está diseñado; nuestro equipo se enfoca en la arquitectura, APIs, base de datos e integraciones.

🏗️ Módulo 1 — Autenticación, Empresas y Onboarding

  • Auth seguro con Supabase (email, magic link, sesiones)
  • Onboarding multi-paso: preguntas iniciales, setup IA, configuración del perfil
  • Gestión de empresas: datos fiscales, sedes, miembros con roles (Propietario, Admin, Trabajador) y áreas personalizadas
  • Sistema de permisos granulares por miembro y módulo
  • Integración Stripe: planes Básico, Pro, Premium, Enterprise con trial de 7 días

🤝 Módulo 2 — CRM (Embudo Comercial)

  • Etapas del embudo: Nueva consulta → Visita comercial → Presupuesto enviado → Aceptado → Cancelado
  • Tarjetas de oportunidad con cliente, sede, productos, montaje, fechas, imágenes y documentos
  • AI Chat que extrae información de la descripción del proyecto y rellena la tarjeta comercial automáticamente
  • Generación de propuesta PDF con imágenes, presupuesto y condiciones; enlace público para el cliente
  • Al aceptar una oportunidad, la IA descompone el trabajo en tareas y subtareas por tablero Kanban automáticamente
  • Revisión humana previa antes de lanzar las tareas

📋 Módulo 3 — Tableros Kanban (Floow)

  • Tablero principal con columnas: Pendiente → Revisión → Aprobado → Oficina Técnica → Producción → Montaje → Entrega
  • Sub-tableros personalizados (Iluminación, Lacado, CNC, Diseño, Control de Calidad, Tercerizado, etc.) vinculados al tablero padre
  • Tareas y subtareas creadas por IA con asignación a usuario, tablero y columna correctos
  • Calendario integrado con cards movibles por fecha de entrega
  • Mensajes internos por proyecto, menciones @miembro y notificaciones

👥 Módulo 4 — Clientes y Proveedores

  • Gestión de clientes: datos fiscales, contactos, sedes con enlace Google Maps, historial de proyectos, presupuestos y facturación
  • OCR para lectura de documentos fiscales y autocompletado de datos
  • Enlace público del cliente para consultar propuestas y albaranes
  • Gestión de proveedores: ficha, pedidos, sistema de solicitud por email con formulario público
  • Proveedor recibe email con info, responde presupuesto/factura; IA extrae y estructura precios y materiales de PDFs/Excel

🤖 Módulo 5 — AI Chat Sign + Generación de Proyectos

  • Chat IA especializado en rotulación (Claude Haiku) disponible en todos los módulos clave
  • Puede crear contactos, propuestas, miembros y escribir en la base de datos según el contexto
  • Generación de proyectos visuales: mockup, fotomontaje/render a partir de sugerencias o selección de producto
  • PDF descargable del proyecto generado, guardado en portal del cliente
  • Integración Fal.ai / Freepik Flux para generación de imágenes

📊 Módulo 6 — Dashboard Master Admin + Analíticas

  • Dashboard con métricas globales: uso de plan, miembros, espacio GB, tableros, créditos IA
  • Estadísticas: nuevos registros, usuarios activos, bajas, interés en productos, uso diario
  • Sentimiento diario (positivo, neutro, negativo), problemas y necesidades principales
  • Cron job que analiza con DeepSeek todas las acciones diarias de las empresas y genera resúmenes, señales y sugerencias para la base de conocimientos
  • Sugerencias automáticas de contenido para KB, preguntas sin respuesta detectadas

⚙️ Módulo 7 — Infraestructura, APIs e Integraciones

  • Backend en Node.js / Next.js API Routes + Supabase (PostgreSQL + RLS + Storage)
  • Deploy en Vercel con variables de entorno seguras
  • Stripe (suscripciones, billing, webhooks), Brevo (email transaccional, notificaciones), Stripe pricing plans
  • Claude Haiku (chat IA), DeepSeek (analíticas), OpenAI Vision / Claude Vision (OCR), Flux.ai / Freepik (imágenes)
  • Documentación de todas las APIs entregada al finalizar

⚠️ No incluye

  • Desarrollo o modificaciones del frontend (ya diseñado por el cliente)
  • Costos de APIs de terceros (OpenAI, Anthropic, Stripe, Brevo, Supabase, Vercel) — corren por cuenta del cliente
  • Diseño de logotipo, branding o identidad visual
  • Módulos fuera del flujo entregado (fases siguientes se cotizan por separado)
