Marketing VIP® Entendimiento
Tu proyecto, nuestra visión

manameli.com necesita evolucionar hacia una experiencia web que no solo luzca mejor, sino que cargue más rápido, posicione mejor en Google y sea completamente editable por el equipo sin tocar código.

🔍 Lo que identificamos

La mayoría de los sitios web actuales están construidos sobre plataformas que priorizan la facilidad de uso sobre el rendimiento: WordPress con plugins pesados, builders visuales que generan código bloatware, JavaScript excesivo que bloquea el render. El resultado es un sitio que pasa Google PageSpeed con 60-70 puntos, carga en 3-5 segundos y pierde posiciones frente a competidores más ágiles. Astro.js resuelve esto de raíz: genera HTML estático por defecto, envía cero JavaScript donde no se necesita, y produce los sitios más rápidos del ecosistema web moderno. El objetivo es claro — un sitio con 95+ en todas las métricas de PageSpeed, ranking SEO superior y un panel de administración que le permita al equipo de Manameli actualizar textos, imágenes, Pixels de Meta/Google y configuraciones SEO de forma autónoma.

Reto 01

Performance actual limitada

Los sitios tradicionales raramente superan 80 puntos en PageSpeed. Astro.js genera HTML estático puro que alcanza 95-100 de forma consistente, reduciendo el tiempo de carga a menos de 1 segundo.

Reto 02

SEO técnico deficiente

Core Web Vitals (LCP, FID, CLS) son factores de ranking directo en Google. Un sitio que falla en performance pierde posiciones orgánicas sin importar qué tan bueno sea el contenido.

Reto 03

Dependencia total del desarrollador

Cambiar un texto, subir una foto o instalar un Pixel de Meta no debería requerir un ticket de soporte. El CMS headless integrado permitirá autonomía total al equipo, sin tocar código.

Reto 04

Tracking y marketing desconectado

Pixel de Meta, Google Tag Manager, Analytics y otras herramientas de marketing deben gestionarse fácilmente desde el panel, sin editar archivos de configuración ni código fuente.

🚀 Desarrollo en Astro.js · Performance 95+

  • Rediseño completo del sitio en Astro.js con arquitectura de componentes reutilizables
  • Generación de HTML estático (SSG): carga instantánea, sin JavaScript innecesario
  • Imágenes optimizadas con el componente nativo de Astro: WebP automático, lazy loading y tamaños responsivos
  • CSS crítico inlineado y estilos cargados de forma asíncrona para eliminar render-blocking
  • Fuentes web optimizadas con font-display:swap y preload selectivo
  • Auditoría final en Google PageSpeed Insights: objetivo 95+ en Performance, Accessibility, Best Practices y SEO

🔎 SEO Técnico Superior

  • Estructura semántica HTML5 con jerarquía de headings correcta (H1→H2→H3)
  • Meta tags dinámicos: title, description y Open Graph por página desde el CMS
  • Sitemap XML generado automáticamente con cada build
  • robots.txt configurado y canonical tags en todas las páginas
  • Datos estructurados (Schema.org) para potenciar la presencia en resultados de búsqueda
  • URLs amigables, limpias y optimizadas para SEO
  • Core Web Vitals optimizados: LCP < 1.5s, CLS < 0.1, FID/INP mínimo

⚙️ Panel de Administración · CMS Headless

  • Integración con Decap CMS (antes Netlify CMS) o TinaCMS: interfaz visual para editar contenido sin código
  • Gestión de textos: títulos, párrafos y llamados a la acción editables desde panel visual
  • Gestión de imágenes: subida y reemplazo de fotos con optimización automática al guardar
  • Gestión SEO por página: meta title, meta description y Open Graph editables por página
  • Gestión de Pixels y scripts de tracking: Meta Pixel, Google Analytics, Google Tag Manager — activación/desactivación y configuración de IDs desde el panel sin tocar código
